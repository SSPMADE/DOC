The style of the text on your web pages is very important to the overall design of your website. You will want the text style to meet your needs. There are physical type styles that you can use when formatting characters. The following is a list of the most popular physical type styles:

<B> Bold Text </B>

<I> Italic Text </I>

<TT> Fixed width font </TT>

<U> Underline Text </U>

<SUB> Subscript </SUB>

<SUP> Superscript </SUP>

You can also change the font of the characters. The default font type for HTML documents is Times New Roman. I personally don't like that font type so I designated arial as the base font type for my website. The <FONT></FONT> tags can be used with the FACE, SIZE, and COLOR attributes to change the font type of characters to be displayed by the web browser. FACE must equal the name of the font, such as FONT="arial". SIZE must equal the size of the characters in points (PT), such as SIZE=2pt. COLOR must equal the hexadecimal color value of the characters, such as COLOR=#000000. For example,

     <FONT FACE="TIMES NEW ROMAN" SIZE=4PT COLOR=#000080>"The Programmer's Domain"</FONT>, would produce the following:

     "The Programmer's Domain"

Example sizes of the point size values from 1 to 7:

1PT : 1 point arial type text
2PT : 2 point arial type text
3PT : 3 point arial type text
4PT : 4 point arial type text
5PT : 5 point arial type text
6PT : 6 point arial type text
7PT : 7 point arial type text


NOTE: In order for your specfied font to override the default font displayed by a web browser, the visitor to your site must have your specified font saved on his computer. For example, if I chose to use the font type ocra as the font type for text on one of my web pages, the visitor to my web page must have the ocra font on his computer; otherwise, he will see the default font type.
