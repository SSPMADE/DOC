[name]

  • HTML - short for HyperText Markup Language hypertext
  • hyperlink - text or image in a web page document that links to another document
  • element - a fundamental component of the structure in a document
  • tag - used to denote various elements in a document; it signals a command or instruction for a web browser; tells a web browser specifically how to display a document
  • attribute - additional information included inside the start tag; gives a command to a web browser telling what kind of operation is required
  • URL - [Uniform Resource Locator] - the address to any website that is part of the world wide web
Every HTML document should contain the basic HTML tags. The head contains the title, and the body contains the elements that are made up of text, images, paragraphs, tables, lists, and other elements. Not every browser supports all HTML tags; however, if the tag is not supported, the browser simply ignores it. HTML is also not case sensitive, which means is the same as or . The following shows the required elements of a document. Open your text editor, type the code in, save it with a .html extension, and you will have your first web page created. You can view your web page document by opening it with a web broswer.
<HTML> <HEAD> <TITLE></TITLE> </HEAD> <BODY> </BODY> </HTML> If you opened your first web page in your browser, you should see a blank white or gray screen. This should make sense because you have not provided any elements between the body tags of your document. By default, web browsers display text in black, hyperlinks in blue, and the background in white or gray. HTML allows you to change these options. The following elements are manipulated in the body tag of a document.

BGCOLOR="Background color"
TEXT="Text displayed in the browser"
LINK="Hyperlink color"
VLINK="Visited Hyperlink color"
ALINK="Active Hyperlink color"

Example: For a black background, silver text, and white hyperlinks, enter <BODY BGCOLOR="#000000" TEXT="#9690CC" LINK="#FFFFFF"> Important Note:
The six digit number and letter combinations represent the amount of RGB (red, green, blue) color as hexadecimal values from 00 to FF. 000000 is black (no color), FFFFFF is white (fully saturated). Finding a specific color using these number and letter combinations can be cryptic.

There are millions of different shades of colors out there. A good image editing program such as Windows Paint will provide you with the hexadecimal HTML value of a color you choose with its color picker. You should download a good paint program or get your hands on a copy of one because these programs are also great for creating your own images for your web pages, as you will find out later.

You can use a background image as the background of your document instead of a color. The BACKGROUND attribute allows you to use an image as the background. For example, if you have an image named SSP001.png on your computer in the same directory as your HTML file, you could use the following code: <BODY BACKGROUND="SSP001.png"> Now, you have created your first web page document.
