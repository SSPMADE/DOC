There may be times when you will need to organize your information or images in a list format. HTML supports the following 3 list types: unumbered, numbered, and definition lists.

Unumbered lists are usually displayed with bullets, which depict each new line of the list. Follow these steps to create an unumbered list:

     1 - Start with the opening unumbered list tag <UL>
     2 - Enter the list item tag <LI> followed by the item to be listed.
     3 - Continue to list items with the list item tag
     4 - End the unumbered list by using the closing tag </UL>

NOTE: The TYPE attribute can be used in the opening <UL> tag to specify a SQUARE or CIRCLE shaped bullet. If no bullet is specified, the default solid disk shape is used. Also note that no closing LI tag is needed when entering list items.

Numbered lists are coded identical to the unumbered list except for the opening and closing tags: <OL></OL>.

Definition lists can be used for two purposes. If you want to list terms and definitions, you can use alternating definition tags <DT> and definition data tags <DD>. Please note that you must use closing tags when you use <DT> and <DD>. You can also use a definition list if you have created a custom bullet that you would like to use in a list that looks much like an unumbered list. You would then use only <DD> opening and closing tags to list the bullet images and items of your choice. To begin and end a definition list, you must use the <DL> start and closing tags. The following are some examples of lists with their corresponding source code:

Example 1: Unumbered list
<ul> <li>Example Text 1</li> <li>Example Text 2</li> <li>Example Text 3</li> <li>Example Text 4</li> </ul>
This document would produce this:
  • Example Text 1
  • Example Text 2
  • Example Text 3
  • Example Text 4


Example 2: Ordered list
<ol> <li>Example Text 1</li> <li>Example Text 2</li> <li>Example Text 3</li> <li>Example Text 4</li> </ol>
This document would produce this:
  1. Example Text 1
  2. Example Text 2
  3. Example Text 3
  4. Example Text 4


Example 3: Definition list - terms and definitions
<dl> <dt><em>Word 1</em></dt> <dd>This corresponds to the meaning of word 1.</dd> <br> <br> <dt><em>Word 2</em></dt> <dd>This corresponds to the meaning of word 2.</dd> <br> <br> <dt><em>Word 3</em></dt> <dd>This corresponds to the meaning of word 3.</dd> <br> <br> <dt><em>Word 4</em></dt> <dd>This corresponds to the meaning of word 4.</dd> </dl>
This document would produce this:
Word 1
This corresponds to the meaning of word 1.


Word 2
This corresponds to the meaning of word 2.


Word 3
This corresponds to the meaning of word 3.


Word 4
This corresponds to the meaning of word 4.

We are now ready to talk about tables. In my opinion, tables are the most important aspect of the HTML language. You can design an entire web page with the whole blueprint grid of the page being one single large table. Tables allow you to place images, objects, and text in the exact position that you specify.
