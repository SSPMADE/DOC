A table can be used to present tabular information such as schedules, statistics, etc., but a table can also be used for organizing complex web pages. If you were to look at the source code to the main section of my site, you would notice that the entire design of my site [http://www11.brinkster.com/warenet/index.htm] is based on one huge table. A table lets you make use of rows and columns so you can specify precisely where you want to place text, images, and objects on your pages. The following describes the table elements and attributes you may use while creating tables:

Table Elements and Corresponding Attributes

<TABLE></TABLE>
     Defines a table in HTML.
*** Attributes ***
BORDER="" - specifies the thickness of the border of the table. 0 = no border; 1 = default; the higher the number, the thicker the border.
CELLPADDING="" - specifies the thickness of the area inside the contents of a cell; the higher the number, the thicker the area.
CELLSPACING="" - specifies the thickness of the area outside the contents of a cell; the higher the number, the thicker the area.
WIDTH="" - specifies the width of the table. You can use pixel values such as WIDTH="500", or you can use percentage values such as WIDTH="100%". Percentages are based on the screen resolution of the computer's display.
BGCOLOR="" - specifies the color of the background of the table; you must use hexadecimal HTML color values as usual.
BACKGROUND="" - specifies an image to be used as the background of the table.

<CAPTION></CAPTION>
Defines a title to be used for the table positioned above the table, by default. You can use the attribute ALIGN="BOTTOM" to place the caption below the table.

<TH></TH>
Defines a table header cell. By default, the text is centered and is of bold type.

<TR></TR>
Defines a table row within the table.

<TD></TD>
Defines a table data cell within the table row.
*** Attributes ***
ALIGN="" - [LEFT, CENTER, RIGHT] - specifies the horizontal alignment of the contents of a cell.
VALIGN="" - [TOP, MIDDLE, BOTTOM] - specifies the vertical alignment of the contents of a cell.
COLSPAN="" - specifies the number of columns a cell will expand.
ROWSPAN="" - specifies the number of rows a cell will expand.
WIDTH="" - specifies the width of the cell. (pixel or percentage values)
HEIGHT="" - specifies the height of the cell. (pixel of percentage values)
BGCOLOR="" - specifies the color of the background of the cell; you must use hexadecimal HTML color values as usual.
BACKGROUND="" - specifies an image to be used as the background of the cell.

     <th bgcolor=#000000><font size=2pt face="arial" color=#FFFFFF>Heading</font></th>
</tr>
<tr>
     <td align="center"><font size=2pt face="arial">Text A</font></td>
     <td align="center"><font size=2pt face="arial">Text B</font></td>
</tr>
<tr>
     <td align="center"><font size=2pt face="arial">Text C</font></td>
     <td align="center"><font size=2pt face="arial">Text D</font></td>
</tr>
</table>

</body>
</html>
