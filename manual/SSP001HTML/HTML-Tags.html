HTML uses tags, the more tags you know, the better you will be able to customize your web pages.
HTML tags contain a left angle bracket ( < ), a tag name, and a right angle bracket ( > ).
Tags normally have a start and end tag. The end tag is identical to the start tag with a backslash ( / ) preceding the text between the brackets.
Some elements may include additional instructions in the start tag, called attributes. Attributes give additional information about the element to the web browser.

The Basic HTML Tags:

<HTML></HTML>
This element simply notifies the web browser that this document contains HTML-coded information. This is also indicated by the .html or .htm extension.

<HEAD></HEAD>
This element identifies the beginning of your HTML document. It contains the title tag and could possibly contain other scripting language code such as Java, Javascript, style sheets, etc.

<TITLE></TITLE>
This element specifies the text to be displayed in the title bar of the web browser. The title is not displayed within the content area of the broswer. You should make the title of your site very descriptive and meaningful because it is what is used if someone bookmarks your site.

<BODY></BODY>
This element contains the content of your document that is to be displayed in the content area of a web browser.

<PRE></PRE>
This is the preformatted text tag. Unlike word processing documents, word wrapping is default (unless otherwise instructed) in a HTML document and multiple spaces are reduced to one single space. This tag makes spaces, line breaks, and tabs appear in the same location as it does in the source file document. The pre tag can also be used with the WIDTH="#" attribute to specify how many characters are displayed on each line.

<HR>
This tag produces a horizontal rule that extends across the content area in a web browser. It is useful for seperating and organizing information. An alternative and better idea would be to create an image of your own that matches your site that will serve the same purpose. You can use two attributes with this tag. The SIZE attribute specifies the line thickness in pixels. The WIDTH attribute specifies the line width in percentage of the screen or the line width in pixel values. <BR>
Using this tag causes a line to break with no extra space. It is the same as pressing enter in a word processing document; it simply breaks the line and begins a new one.
